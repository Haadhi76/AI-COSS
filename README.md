# INNATEAI AI Chief of Staff

This repository contains the INNATEAI AI Chief of Staff system.

## Project Structure

- `frontend/`: React/Vite application.
- `backend/`: FastAPI Python application acting as a secure proxy for the Anthropic Claude API.

## Requirements
- Node.js (for frontend)
- Python 3.11+ (for backend)

See the `CLAUDE.md` for architecture details.
